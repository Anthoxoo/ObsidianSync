## Fonctionnement LLM : 
- Objectif : prédire le prochain mot du texte
- Pour améilorer efficacité IA --> On ajoute un très grand nombre de paramètres.
- Token --> Unité minimale compréhensible par l'IA, ca peut etre un mot, une partie de mot, de la ponctuation etc...
- LLM cherche constamment le prochain token

## Limites des LLM
- Pas de compréhension réelle
- Biais positifs ou négatifs
- Hallucinations --> peut inventer des choses qu'il n'a jamais apprises, données proposées sur internet fausses.
- Dépendant des données d'entrainement
