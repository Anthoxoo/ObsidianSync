1)  Présentation : Forme des étudiants au métier d'ingénieur dans la conception et le développement de logiciels informatiques.

2) Débouchés + Graph : 
- Développement logiciel
- Ingénieurie des systèmes
- Chef de projet
Possibilité de poursuivre les études avec un doctorat mais très minoritaire.

3) Qualités requises :
- Motivation 
- Travail d'équipe
- autonomie 
- rigueur
- Formation recommandée : Licence informatique

4) Présentation des UE :
- Première année :
	- premier semestre :
		- ( dire a l'oral que tronc commun avec IHM sur le premier semestre )
	- second semestre :
		- Interaction homme-machine
		- Modèle architecture 
		- Gestion de projet
		- Developpement web
- Deuxième année :
	- Ingénierie des modèles et systèmes.
	- Sécurité et optimisation des logiciels.
	- Vérification et analyse des programmes informatiques
	- Poursuite des UE de première année
- Stage ou alternance obligatoire chacune des deux années.

5) Frontend :
- Partie visible d'un logiciel / site web
- Languages utilisés : Javascript, HTML, CSS
- Objectif : améliorer l'expérience utilisateur

6) Backend :
- Partie invisible d'un logiciel / site web
- Gestion de la logique et traitement des données
- Traiter les requetes de l'utilisateur.

- Frontend et backend intéragissent entre eux
- les deux sont nécéssaires au bon fonctionnement du logiciel / de la page web.
