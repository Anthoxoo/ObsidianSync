Circuit numérique --> manipulation de 0 et 1 ou encore respectivement Faux et Vrai
3 opérations fondammentales :
- **ET** : peut s'écrire a.b
- **OU** : peut s'écrire a + b
- **Négation** : peut s'écrire : a barre
Rappel : Table de vérité --> peut se faire avec toutes les opération, on met une colonne a une colonne b et une colone ab qui va etre le résultat de l'opération avec la valeur toutes les combinaisons de a = 0 puis 1 et pareil pour b.

--> possibilité de faire des tables de vérité plus complexe avec des fonctions, ex : f(a,b,c) = a + b.c barre *Voir moodle pour table de vérité de la fonction et essayer de la refaire.*
Lorsque la table est terminée, on peut en déduire une équation pour chaque évenement

**Exceptions** dans l'algebre de boole : l'absorbtion : a + a barre.b = a + b
lois de morgan : (a + b) barre = a barre. b barre, (a.b) barre = a barre + b barre

On a le droit de factoriser les termes de l'algebre ainsi que les simplifier etc...*voir moodle*
