# Histoire de l’informatique – Sciences du numérique (L1)

## Préambule
- Informatique = science du traitement automatique de l’information  
- Ordinateur = machine électronique programmable  
- Programme = suite d’instructions  
- Schéma de base : **Entrées → Traitement → Sorties**

---

## Les fondements

### Avancées théoriques
- **Numération** : passage du calcul manuel aux systèmes positionnels (zéro, chiffres arabes).  
- **Algorithmique** : ensemble fini d’instructions pour résoudre un problème.  
- **Logique booléenne (Boole)** : vrai/faux → base du calcul informatique.  
- **Machine de Turing (Turing, 1936)** : modèle théorique d’un ordinateur (ruban + tête de lecture/écriture).  

### Avancées technologiques majeures
- **Carte perforée (Jacquard, Hollerith)** : première idée de programmation par données externes.  
- **Tube à vide** : permet l’électronique (interrupteur ON/OFF).  
- **Transistor (1947)** : plus petit, rapide, fiable → remplace les lampes → révolution des ordinateurs.  
- **Circuit intégré** : plusieurs transistors sur une même puce.  
- **Microprocesseur (1971)** : processeur complet sur une puce → naissance du PC.

---

## Époques de l’informatique

### 1. Avant l’ordinateur
- Abaque, boulier → aides au calcul manuel.  
- Pascaline (Pascal) → addition/soustraction mécaniques.  
- Leibniz → multiplication/division.  
- Babbage & Ada Lovelace → première machine programmable théorique.  

### 2. Premiers ordinateurs (années 1940-1950)
- Machines électromécaniques (Zuse, Stibitz).  
- **Colossus** : utilisé pour casser les codes (cryptanalyse).  
- **ENIAC** : premier grand calculateur électronique (lampes).  
- **Architecture de Von Neumann** :  
  - Mémoire unique contenant données + instructions  
  - Unité de calcul, contrôle, entrées/sorties  
  - → Modèle encore utilisé aujourd’hui.  

### 3. Deuxième génération (années 1950-60)
- **Transistor** : ordinateurs plus fiables, plus petits.  
- Apparition des **langages de programmation** (Fortran, Cobol).  
- Début de l’informatique commerciale (IBM).  

### 4. Troisième génération (années 1960-70)
- **Circuits intégrés** : miniaturisation → baisse des coûts.  
- Systèmes d’exploitation multi-tâches.  
- Premiers réseaux (ARPANET → Internet).  

### 5. Quatrième génération (années 1970-90)
- **Microprocesseur** (Intel 4004, 1971) → naissance du micro-ordinateur.  
- Apple, IBM PC → démocratisation.  
- Début des logiciels libres, UNIX.  
- Développement des réseaux et du courrier électronique.  

### 6. Cinquième génération (depuis 1990)
- **Internet grand public** : Web, réseaux sociaux.  
- **Mobilité** : ordinateurs portables, smartphones.  
- **Puces multi-cœurs** pour plus de puissance.  
- Vers l’**informatique ubiquitaire** : cloud, IA, objets connectés.  

---

## Conclusion
- L’informatique a évolué :  
  - Calcul manuel → mécanique → électronique → numérique → ubiquitaire  
- Inventions majeures :  
  - **Logique booléenne** : base du calcul.  
  - **Machine de Turing** : modèle théorique.  
  - **Transistor** : miniaturisation et fiabilité.  
  - **Architecture de Von Neumann** : base des ordinateurs modernes.  
  - **Microprocesseur** : ordinateur personnel.  
- Aujourd’hui : informatique omniprésente, réseau mondial, intelligence artificielle.
